# Refine & Deploy - Web Agency Portfolio

High-performance web agency portfolio built with Astro 5.x and Tailwind CSS v4.

## Tech Stack
- Astro 5.x (Static Site Generator)
- React (Interactive islands only)
- Tailwind CSS v4
- Vercel (Hosting)

## Development
```bash
npm run dev      # Start dev server
npm run build    # Build for production
npm run preview  # Preview build locally
```
